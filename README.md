# دفتر العناوين — Jawwal Address Book

## 🚀 تشغيل المشروع

### المتطلب الوحيد
- [Node.js](https://nodejs.org/) — حمّل النسخة LTS

### الخطوات (3 فقط)

```bash
npm install
npm run dev
```

افتح المتصفح على: **http://localhost:5173**

---

## 📁 الملفات

```
address-book-app/

├── package.json
└── src/

    └── AddressBookApp.jsx   ← كامل التطبيق
```
